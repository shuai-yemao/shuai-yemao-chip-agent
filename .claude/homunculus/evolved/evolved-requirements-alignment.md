# 需求对齐引导（进化技能）

## 触发条件
当开始新功能开发或需求实现时

## 能力摘要
坚持需求对齐层流程：grill-me → 约束梳理 → 六源审查 → 清单确认 → to-issues 拆解

## 应用规则
1. 新需求第一步永远是 `requirements-alignment` Workflow
2. 对话式交互，不要跳过用户确认
3. 输出 chain 元数据指导后续步骤
4. 5 步走完前不执行
5. 验收清单需要用户最终判决（verdict: pending → passed/failed）

## 关联技能
- requirements-alignment Workflow
- grill-me / grill-with-docs / to-issues（需求拆解技能）
- safety-layer（安全预检）
